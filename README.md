# Ethic Hawks – Forensic Governance & Risk Architecture

The official website of **Ethic Hawks**, a Cape Town‑based forensic governance consultancy.  
Built around the proprietary **Wilbur Method™** – a forensic architecture that diagnoses and excises Institutional Sepsis through the Closed Loop of Truth.

## What this repository contains

- `index.html` – The main website: all sections, responsive design, and the core forensic framework.
- `terms.html` – Terms of Service (legal agreement for site use).
- `privacy-policy.html` – Privacy Policy (POPIA‑compliant, essential‑only cookies, no tracking).
- `paia-manual.html` – PAIA Manual (Promotion of Access to Information Act guide).
- `images/` – Folder for your hawk crest logo (`hawk-crest.png`) and founder photo (`founder-photo.jpg`).

## How to deploy

1. Upload all files to your GitHub repository.
2. Go to **Settings → Pages**, select the `main` branch and root folder, then **Save**.
3. Your site will be live at `https://yourusername.github.io/repository-name/`.
4. (Optional) Point your custom domain `www.ethichawks.co.za` to GitHub Pages.
